## Functional View

```

//Function returning just a view with no other power:
View = (function(props) {
	return <View />
})();

```

```

//Returning a ViewComponent with lots of powers:
ViewComponent = React.createComponent({
	render() {
		return <View />
	}
});

````

## 3 Ways of Creating React Components:
- Classical Way (ES5 way)
- Classy Way (ES6 way)
